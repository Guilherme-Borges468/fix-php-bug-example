# Exemplo de Correção de Bug em PHP

Projeto simples para mostrar antes/depois de uma correção.

## Arquivos

- `broken.php` → código com bug (foreach usando variável errada)
- `fixed.php` → versão corrigida

Você pode usar esse tipo de exemplo para mostrar ao cliente que sabe:

- encontrar problema
- explicar o motivo do erro
- entregar o código corrigido
